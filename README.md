# 2章

- `tsc --init` で `tsconfig.json` を作製する。 このファイルは ts ファイルのコンパイル時のオプションを記述する
- `tsconfig.json` の `strictNullChecks` は `true` がお勧めのよう
- typescript では `null` は用いずに `undefined` に統一するのがよい。 ただし、書籍内では紹介目的で `null` を用いることも
